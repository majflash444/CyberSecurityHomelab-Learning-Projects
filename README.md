# Cybersecurity Homelab & Learning Journey

## Overview

This repository documents my cybersecurity learning journey through hands-on labs, virtual environments, and structured documentation. The goal is to build practical skills in Linux, networking, system administration, and security fundamentals while developing a professional portfolio.

I am currently building a virtual homelab using VirtualBox and Ubuntu Server, and documenting all progress using Git and GitHub.

---

## 🎯 Goals

- Build a strong foundation in Linux system administration
- Learn networking fundamentals (TCP/IP, DNS, ports, routing)
- Gain hands-on experience with cybersecurity tools
- Develop a structured, professional documentation workflow
- Transition from beginner → job-ready cybersecurity skillset

---

## 🧱 Learning Roadmap

### Phase 1: Foundations (Current)
- Git & GitHub fundamentals
- VirtualBox setup
- Ubuntu Server installation
- Linux command-line basics
- File system navigation
- Users & permissions

### Phase 2: Networking Basics
- IP addressing
- DNS & DHCP
- TCP vs UDP
- Packet analysis
- Basic network troubleshooting

### Phase 3: Security Tools
- Nmap scanning
- Wireshark analysis
- Log analysis
- Basic vulnerability scanning

### Phase 4: Homelab Expansion
- Multiple virtual machines
- Internal networking (host-only / bridged)
- Simulated attack & defense scenarios
